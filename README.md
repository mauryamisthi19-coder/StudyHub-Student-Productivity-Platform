# StudyHub – Student Productivity Platform

StudyHub is a responsive student productivity platform developed using HTML, CSS, and JavaScript. The application helps students manage tasks, notes, study goals, attendance, quizzes, and productivity sessions from a single dashboard.

## Features

* Dashboard with statistics
* Task Manager
* Notes Manager with search
* Pomodoro Timer
* Attendance Tracker
* Quiz App
* Study Goal Tracker
* Dark Mode
* Live Clock and Greeting
* Local Storage Support
* Mobile Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## Project Structure
